# PolicyForge Prototype

Interactive desktop prototype for the PolicyForge AI governance control center.

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:4173`.

## Build

```bash
npm run build
```

## Demo path

1. Review and approve the Candidate Screening Agent.
2. Run the governed candidate action and complete human review.
3. Inspect Decision Capsule `PF-2841`.
4. Replay decisions under Recruitment Policy v1.5.
5. Create Recall `RC-017`.

Use **Reset demo** in the application header to return to the initial state.
